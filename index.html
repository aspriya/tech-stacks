How to use

  1. Open a stack page and download the markdown files you need.
  2. Drop them into your project under docs/.
  3. Point your AI agent at docs/, or attach the files when starting a session.
  4. The agent now knows the stack, conventions, and what not to do.
# From the root of your new project
mkdir -p docs
curl -L -o docs/tech-stack.md \
  https://<your-username>.github.io/tech-stacks/nextjs-cloudflare/tech-stack.md

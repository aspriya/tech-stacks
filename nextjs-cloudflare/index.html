Grab everything in one go

Pull all four files into docs/ from your new project's root:

mkdir -p docs
BASE="https://aspriya.github.io/tech-stacks/nextjs-cloudflare"
curl -L -o docs/tech-stack.md             "$BASE/tech-stack.md"
curl -L -o docs/cloudflare-automation.md  "$BASE/cloudflare-automation.md"
curl -L -o docs/payment-processing.md     "$BASE/payment-processing.md"
curl -L -o docs/tip-jars-and-donations.md "$BASE/tip-jars-and-donations.md"

Or PowerShell: 

New-Item -ItemType Directory -Force -Path docs | Out-Null
$base = "https://aspriya.github.io/tech-stacks/nextjs-cloudflare"
Invoke-WebRequest "$base/tech-stack.md"             -OutFile docs/tech-stack.md
Invoke-WebRequest "$base/cloudflare-automation.md"  -OutFile docs/cloudflare-automation.md
Invoke-WebRequest "$base/payment-processing.md"     -OutFile docs/payment-processing.md
